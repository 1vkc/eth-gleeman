# Gleeman

Using LLM to transfer raw callbacks to natural language explanation.

## Getting Started

```
pip install -r requirements.txt
```

## Demo

### Adhoc

```sh
python adhoc_demo.py
```

### Web UI

```sh
cd demo
streamlit run demo.py
# or
./run.sh
```
